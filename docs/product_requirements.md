# Product Requirements

## MVP

### Roles
- manager
- superuser

### Survey fields
- visit datetime
- table number
- 4 yes/no answers
- optional comment

### Features
- manager can create survey
- manager can view reports for own cafe
- superuser can view reports for one cafe or all cafes
- reports support period comparison

### Notes
- paper surveys are entered manually by managers
- each survey belongs to one cafe
